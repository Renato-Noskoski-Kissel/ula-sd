# ULA do MIPS - Trabalho Prático de Sistemas Digitais
O projeto consiste no desenvolvimento de uma ULA MIPS (Unidade Lógica e Aritmética) capaz de executar as instruções padrão (adição, subtração, and, or e set on less than) e outras duas instruções extras: uma para realizar a multiplicação entre dois inteiros e outra para efetuar a divisão inteira, com saída para o quociente e o resto da operação.

Os principais componentes que serão utilizados são full adders, half adders, complemento de dois, blocos de AND e OR bit a bit, além do multiplicador e o divisor, garantindo uma ULA funcional e completa.
